# Dotfiles Ubuntu WSL
## Prerequisites
- Nerd Fonts
- zsh
- zplug
- Starship
## Installation
### zplug
Zsh Plugin Manager
```bash
curl -sL --proto-redir -all,https https://raw.githubusercontent.com/zplug/installer/master/installer.zsh | zsh
```
### Starship
```bash
curl -sS https://starship.rs/install.sh | sh
```
